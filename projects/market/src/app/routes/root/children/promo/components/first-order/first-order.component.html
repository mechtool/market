Акция 10% на первый заказ

С 1 сентября по 31 октября для пользователей сервиса 1С:Торговая площадка действует скидка 10% на первый заказ от поставщиков, участвующих в акции, в категориях «Товары для школы и офиса», «Продукты, напитки, табак», «Текстиль для детей», «Изделия из металла» и «Товары для животных». Чтобы посмотреть товары, перейдите в соответствующий магазин поставщика. Цены в категории указаны без скидок. Чтобы воспользоваться скидкой, отправьте заказ поставщику, участвующему в акции, ваш заказ будет доставлен с пометкой "скидка". Скидка к заказу будет применена поставщиком при выставлении счета.

Как применить скидку на заказ?

  1. Выберите популярные товары на текущей страничке или перейдите в магазин поставщика, участвующего в акции, нажав на соответствующий баннер на странице акции.
  2. Укажите нужное количество товара и перейдите в корзину для оформления заказа.
  3. Внесите ваши контакты данные, чтобы менеджер поставщика смог связаться с вами и оформить скидку на заказ.
  4. Все отправленные заказы со скидкой можно увидеть в личном кабинете.

Акция скидка 10% на первый заказ проводится c 1 сентября по 31 октября включительно.
