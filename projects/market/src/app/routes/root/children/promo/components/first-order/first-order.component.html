Акция Скидка 10% на первый заказ

С 1 сентября по 31 октября для пользователей сервиса 1С:Торговая площадка действует акция "Скидка 10% на первый заказ" от Поставщиков, участвующих в акции в категориях «Товары для школы и офиса»,"Продукты питания", "Текстиль для детей", "Изделия из металла" и "Зоотовары ". Чтобы посмотреть товары, перейдите в соответствующий магазин Поставщика. Цены в категории указаны без скидок. Чтобы воспользоваться скидкой отправьте заказ Поставщику, участвующему в акции, ваш заказ будет доставлен с пометкой "скидка 10%". Скидка к заказу будет применена Поставщиком при выставлении счета.

Как применить скидку на заказ?

  1. Перейдите в магазин Поставщика, участвующего в акции, нажав на соответствующий баннер на странице Акции вместо баннера ниже должен быть баннер комус- первый сверху;
  2. Положите нужное количество товара в корзину и перейди в корзину для оформления заказа;
  3. Укажите ваши контакты данные, чтобы менеджер поставщика смог связаться с вами, и оформить скидку на первый заказ;
  4. Все отправленные заказы со скидкой можно увидеть в личном кабинете.

Акция скидка 10% на первый заказ проводится c 1 сентября по 31 октября включительно.
